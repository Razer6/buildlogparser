# buildlogparser Gem

Collection of various parsers for the extraction of information from build and execution logs.

## Tested Program Versions 

* cmake/ctest 3.5.1
* dhrystone (C Version) 2.1 and 2.2
* size (GNU Binutils 2.26.1)
